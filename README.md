# 🧠 Sentiment Analysis on Amazon Reviews

A complete end-to-end NLP project that predicts whether a review is Positive or Negative.

## 🚀 Features
- Text preprocessing using NLTK
- TF-IDF vectorization
- Logistic Regression model
- Flask web app for real-time prediction

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK
- Flask

## 📂 Project Structure
- preprocess.py → data cleaning & preprocessing
- model_train.py → model training & saving
- app.py → Flask backend
- templates/index.html → frontend UI

## 📊 Model Performance
- Accuracy: ~93%
- Used Confusion Matrix, Precision, Recall, F1-score
